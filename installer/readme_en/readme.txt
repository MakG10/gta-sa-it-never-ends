GTA SA: It Never Ends


####################### About #######################
Mission pack GTA SA: It Never Ends contains:
- 8 missions with voiced dialogues + 3 extra missions
- 8 unique jumps
- 8 money bags to collect
- Cover System
#####################################################


################### Installation ####################
Installation process is done by setup file named
"ine.exe". If you have downloaded a file
with different name or an archive with modification
files, then we strongly recommend to redownload it
from an official websie to avoid any futher problems:

http://www.gta-mods.pl/ine
#####################################################


#################### INE Extras #####################
The installer includes 3 extra missions
- Kidnapping
- Da Nang Fags
- Take It Back

The player acts as one of the men of Bill Degorio,
the actions takes place during events from the
basic version of It Never Ends.

You can activate the missions using INE menu (type
"inemenu" in-game).

Source code of the missions can be found on
http://www.gta-mods.pl/
#####################################################


################ Modification files #################
The following paths are relative to the
GTA San Andreas directory.

New files:
- CLEO/ine.cs
- CLEO/INE/*
- CLEO/CLEO_TEXT/INE_(en|pl).fxt - depends on
                                   installed
                                   language version.
- data/Paths/carrec.img:
  * files carrec890.rrr - carrec902.rrr
    (Required for missions: Show Time, Rescue)
- models/txd/ine.txd
- data/models/gta3.img:
  * fbiboss.dff
  * fbiboss.txd

Replaced files:
- data/models/gta3.img:
  * artict1.dff (required for mission "Jump On It")

ATTENTION!
 A note below refers to a small group of modifications
 and it probably doesn't affect you.

 If previously installed modifications,
 or the ones you will install in future
 are using their own *.rrr paths,
 then their naming may conflict with the ones provided in
 "It Never Ends", what may cause problems
 in missions "Show Time" and "Rescue".

 If you want "It Never Ends" to be compatible with
 your other modification, then send an e-mail message
 on the following address:
  kontakt@gta-mods.pl
 An e-mail must have the following template:

 Title:   INE: rrr Paths
 Content: Uncompatible modification:
           XXX downloaded from YYY
          Conflicting path files:
           carrecAAA.rrr, carrecBBB.rrr etc.

 The messages with incorrect template will be ignored.
